API Documentation

Welcome to the API documentation for our data masking service. Our API allows you to integrate data masking functionalities into your applications seamlessly, enabling you to enhance your data security and compliance.

Getting Started

To begin using the API, you need to authenticate using your API key. You can find your API key in your account dashboard after logging in.

Endpoints

  • POST /mask: Masks sensitive data in a given dataset.
  • GET /status: Retrieves the status of a masking operation.

Example Request

curl -X POST https://api.yourservice.com/mask \
-H "Authorization: Bearer YOUR_API_KEY" \
-d '{"data": "Your sensitive data here"}'

Response

{
    "masked_data": "Masked data result"
}
